# Lucky65 v2 VIA Layout (Ported from ZMK Lily58)

This repository contains a VIA Save+Load layout file for Lucky65 v2, ported from my ZMK Lily58 keymap.

## File

- `lucky65v2_zmk_via_layout.json`

## Notes

- Target keyboard matrix: 5x15 (Lucky65 v2)
- `vendorProductId`: `875422727`
- Layers: 4

## Usage

1. Open https://usevia.app
2. Enable **Design tab** in Settings
3. Load the official Lucky65 v2 definition JSON
4. Go to **Save + Load** and load `lucky65v2_zmk_via_layout.json`
